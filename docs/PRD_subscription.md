
# PRD.md — Subscription-based Model

## Terminology
- Account = akun ChatGPT (tidak expired)
- Subscription/Invite = akses ke plan
- Expired = berakhirnya subscription

## Goal
Mengelola akun + lifecycle subscription + usage monitoring + reminder.

## Core Features
- CRUD Accounts
- CRUD Subscriptions (per account bisa banyak)
- Usage per subscription
- Telegram reminder subscription expired

## Success Criteria
- Tidak ada subscription terlewat expired
- Usage tercatat per subscription
